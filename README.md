# 🏖️ リタイアメントナビ

定年前後のお悩みを解決する総合情報サイトです。
年金・健康保険・税金・社会保険など、退職前後に必要な情報をわかりやすくまとめています。

## 機能

- 年金制度の解説（受給開始手続き・受給額計算・受給時期の選択）
- 健康保険・介護保険の切り替え案内
- 税金・確定申告の情報（退職金の税金・各種控除）
- 社会保険・雇用保険の手続き案内
- 年金受給額の簡易計算ツール
- 退職手続きチェックリスト
- 退職後の税金シミュレーター

## 使用技術

- HTML / CSS / JavaScript
- Claude AIを活用して作成

## 対象ユーザー

定年退職を控えた方、退職後の手続きや制度を調べたい方
